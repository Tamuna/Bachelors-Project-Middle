ქვემოთ ჩამოთვლილ ყველა სერვისს, უნდა გავაყოლოთ შემდეგი header-ები:
Content-Type: application/json
X-Requested-With: XMLHttpRequest


(auth)
1. User Registration
request type:
	post
route:
	api/auth/register
body params:
	user_name
	email
	password
	c_password


(auth)
2. Login
request type:
	post
route:
	api/auth/login
body params:
	user_name
	password


ქვემოთ ჩამოთვლილი ყველა სერვისი საჭირეოებს ავტორიზებულ მომხმარებელს.
default-ებთან ერთად უნდა დავამატოთ შემდეგი header:
Authorization
Authorization-ის შესაბამის მნიშვნებლობას აქვს შემდეგიე ფორმატი: "Bearer [ავტორიზაციის შემდეგ მიღებეული token]"
მაგალითი: Authorization: "Bearer eyJ0eXAiOiJKV1QiLCJhbGciOiJSUzI1NiIsImp0aSI6IjcyZjAwNTEwZTQxY"
